HTML TAGS

  1. Anchor Tag: Defines a hyperlink
  2. Article Tag
    Defines an article
  3. Bold Tag:Defines bold text
  4. Audio Tag: Defines embedded sound content
  5. Body Tag: Defines the document's body
  6. Br Tag: Defines a single line break
  7. : Defines a clickable button
  8. Del Tag: Defines text that has been deleted from a document
  9. Div Tag: Defines a section in a document
  10. em Tag:Defines emphasized text
  11. Embed tag: Defines a container for an external application
  12. Footer tag:Defines a footer for a document or section
  13. Form Tag
    : Defines an HTML form for user input
  14. Head Tag: Contains metadata/information for the document
  15. Header tag: Defines a header for a document or section
  16. hr tag: Defines a thematic change in the content
  17. img tag: Defines an image
  18. input tag: Defines an input control
  19. link tag: Defines the relationship between a document and an external resource (most used to link to style sheets)
  20. main tag:Specifies the main content of a document
  21. nav tag:Defines navigation links
  22. p tag: Defines a paragraph
  23. option tag :Defines an option in a drop-down list
    24. etc...
